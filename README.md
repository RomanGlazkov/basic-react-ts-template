# basic-react-ts-template
Basic React/Typescript template to start coding immediately

## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/RomanGlazkov/basic-react-ts-template.git
   ```

2. Install NPM packages
   ```sh
   npm install
   ```

3. Start the project
   ```sh
   npm run start
   ```

4. Enjoy coding!
